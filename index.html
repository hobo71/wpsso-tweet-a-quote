Description

Add Twitter-style quoted text to your content that's easily Tweeted by your readers.

Features an easy "Tweet a Quote" toolbar button in the Visual Editor — along with a simple shortcode for the Text Editor — to create Tweetable quotes quickly and easily.

Uses your existing WPSSO settings to shorten URLs, add a Twitter business @username, and recommend the author's @username after sharing.

Developers and advanced users will appreciate the ability to easily re-style the CSS that WPSSO TAQ uses for the quote and Tweet link.

WPSSO TAQ is fast and coded for performance:

WPSSO and its extensions make full use of all available caching techniques (persistent / non-persistent object and disk caching), and load only the PHP library files and object classes they need, keeping their code small, fast, and light.

WPSSO and its extensions are fully tested and compatible with PHP v7.x (PHP v5.3 or better required).

WPSSO (Core Plugin) Prerequisite

WPSSO Tweet a Quote is an extension for the WPSSO (Core Plugin) — which creates complete and accurate meta tags and Schema markup from your content for social sharing, social media, search / SEO and rich cards.
